Anotações: Grafos

 2022-08-15  resumo grafos 
Conceitos Um grafo é uma estrutura que representa um conjunto de elementos (vértices) e suas relações de interdependência (arestas). ordem = quantidade de vértices no grafo tamanho = ordem + arestas grau de um vértice -> quantidade de arestas incidentes para um vértice sequência gradual de um grafo -> sequência de graus dos vértices de um grafo isomorfismo -> grafos são isomorfos quando possuem a mesma sequência gradual
